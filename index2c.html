COURSE DESCRIPTION

This 160 hour course incorporates JavaScript into the web development process. Students will learn how this language can enhance a webpage, allowing additional interactivity and more precise control of page elements. More advanced techniques used in creating a website and making the content more dynamic will also be taught. Several types of automation tools will be introduced that can further enhance web development and project maintenance.

COURSE OBJECTIVES

Upon successful completion of the course, the student will be able to:

  1. Describe and use JavaScript.
  2. Explain and demonstrate how to programmatically retrieve page content using JavaScript.
  3. Explain and demonstrate how to programmatically manipulate page content using DOM Scripting techniques.
  4. Describe and demonstrate interactivity using the JavaScript event model.
  5. Explain and demonstrate other third-party development tools used in website development.

PROGRAM COMPLETION REQUIREMENTS

Assignments

You will have a total of 10 assignments due in this course. Each assignment is worth an average of 50 points for a total of 500 points. Assignments should be submitted to a new repo within your GitHub account. Each repo should be named Assignment01, Assignment02, etc. You will send me a link to your repo so that I can fork it within Brackets. Late assignments will not be accepted.

Quizzes

You will have a total of 6 concept quizzes due in this course. Each quiz is worth 25 points for a total of 150 points. Quizzes will be administered at the end of class on Thursday evenings (lab) and you will have a total of 30 minutes to complete each quiz. Quizzes will be administered within the Blackboard site for this course.

Final and Final Project

You will have 1 final concepts exam due in this course. The final exam is worth a total of 50 points. The final exam will be administered within the Blackboard site for this course. You will have 1 hour to complete the exam.

You will have 1 final project due in this course. The final project is worth a total of 300 points.
